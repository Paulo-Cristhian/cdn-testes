Listas

A tag HTML <ul> é usada para especificar uma lista não ordenada, que agrupa uma coleção de itens sem ordem numérica. Ao alterar a ordem dos itens da lista, o significado não muda. Normalmente, os itens de uma lista não ordenada são exibidos com um marcador.

  • Modalidade: presencial
  • Para quem: alunos e não alunos
  • Quando: 21/08/2022 às 13h
